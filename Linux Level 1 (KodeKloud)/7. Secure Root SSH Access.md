## <center> Secure Root SSH Access

```
Following security audits, the xFusionCorp Industries security team has rolled out new protocols, including the restriction of direct root SSH login.


Your task is to disable direct SSH root login on all app servers within the Stratos Datacenter.
```

---


Direct Solve:
```sh
ssh-keygen -t rsa -b 2048
for server in tony@stapp01 steve@stapp02 banner@stapp03;
    do ssh-copy-id $server;
done
for server in tony@stapp01 steve@stapp02 banner@stapp03;
    do ssh -t $server "sudo bash -c 'sed -i \"s/^#\?PermitRootLogin.*/PermitRootLogin no/\" /etc/ssh/sshd_config; grep PermitRootLogin /etc/ssh/sshd_config; systemctl restart sshd'"
done
```
